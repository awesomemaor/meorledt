# 🎉 Maor-ledet (המעורלדת)

Welcome to the **Maor-ledet** project! What started as a humorous birthday party concept has evolved into a fully interactive web application. This project showcases modern Front-end development, AI integration, and custom 2D game mechanics.

👉 **[Experience the live site here!](https://maor-site.vercel.app/)**

## ✨ Features

- 🤖 **AI Party Chatbot:** Powered by the **Gemini API**, this smart assistant knows all the lore and details about the party and can chat with users.
- 🕹️ **2D Mini-Game ("Escape the Party"):** A built-in endless runner game inspired by the classic Google T-Rex, featuring custom collisions, assets, and game logic.
- 📱 **Dynamic UI & Animations:** Fully responsive design featuring photo galleries, dynamic content areas ("Entertainment News"), and smooth CSS transitions.
- ⚡ **Optimized Performance:** Built with React and Vite for lightning-fast load times and a great developer experience.
- 🚀 **Continuous Deployment:** Automatically deployed and hosted via Vercel.

## 🛠️ Tech Stack

- **Framework:** React
- **Build Tool:** Vite
- **Environment:** Node.js
- **AI Integration:** Google Gemini API
- **Deployment:** Vercel
